# Llm Routing

Intelligent LLM routing: auto-select best model per request.

## Strategies
- **Cost-optimized**: Route to cheapest model that meets quality threshold
- **Latency-optimized**: Route to fastest available model
- **Quality-optimized**: Route to best model for task type
- **Fallback**: Automatic failover on 429/503

## Supported Providers
OpenAI, Anthropic, Google, Mistral, DeepSeek, Groq, Together
